# 生产环境部署检查清单

在将系统部署到生产环境之前，请确保完成以下所有检查项。

## 环境准备

- [ ] 服务器已安装 Node.js (v14+)
- [ ] 服务器已安装 PM2 (`npm install -g pm2`)
- [ ] 服务器已安装 Nginx（可选，推荐）
- [ ] 服务器防火墙已配置（开放80/443端口）
- [ ] 已获取域名并配置DNS解析

## 配置检查

### 后端配置 (backend/.env)

- [ ] `NODE_ENV` 设置为 `production`
- [ ] `PORT` 配置正确（默认3001）
- [ ] `CORS_ORIGIN` 设置为实际前端域名
- [ ] `DB_PATH` 路径正确
- [ ] `LOG_LEVEL` 设置为 `info` 或 `warn`
- [ ] `LOG_DIR` 路径存在且有写入权限
- [ ] `BACKUP_DIR` 路径存在且有写入权限

### 前端配置 (.env)

- [ ] `REACT_APP_API_URL` 设置为实际后端API地址

## 安全检查

- [ ] `.env` 文件已添加到 `.gitignore`
- [ ] 数据库文件权限设置为 600 (`chmod 600 backend/bearings.db`)
- [ ] 日志目录权限正确
- [ ] 备份目录权限正确
- [ ] 已配置 HTTPS（使用 Let's Encrypt 或其他证书）
- [ ] 已配置 Nginx 反向代理（推荐）
- [ ] 已设置防火墙规则
- [ ] 已禁用不必要的端口

## 数据准备

- [ ] 数据库已初始化 (`npm run init-db`)
- [ ] 已添加实际产品数据
- [ ] 已测试订单创建流程
- [ ] 已设置数据库自动备份（cron任务）

## 构建和部署

- [ ] 前端已构建 (`npm run build`)
- [ ] 构建文件已复制到 backend/public
- [ ] 后端依赖已安装 (`npm install --production`)
- [ ] PM2 配置文件 (ecosystem.config.json) 已检查
- [ ] PM2 服务已启动 (`pm2 start ecosystem.config.json`)
- [ ] PM2 已设置开机自启 (`pm2 startup` 和 `pm2 save`)

## 功能测试

- [ ] 前端页面可以正常访问

- [ ] 管理后台统计数据正确

- [ ] 并发订单处理正常
