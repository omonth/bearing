### 2. 数据分析API
**文件**: `backend/server.js`

新增7个API接口：

#### 综合仪表板数据
```http
GET /api/analytics/dashboard
```
返回所有仪表板数据（一次性获取）

#### 销售趋势
```http
GET /api/analytics/sales-trend?period=day&days=30
```

#### 热销产品
```http
GET /api/analytics/top-products?limit=10&days=30
```

#### 分类销售统计
```http
GET /api/analytics/category-sales?days=30
```

#### 客户地区分布
```http
GET /api/analytics/customer-distribution
```

#### 收入统计
```http
GET /api/analytics/revenue-stats?days=30
```

#### 实时销售监控
```http
GET /api/analytics/realtime-sales
```

## 📊 数据展示

### 统计指标
- **总订单数**: 显示所有订单（不含已取消）
- **总收入**: 显示总销售额
- **平均订单金额**: 客单价
- **产品总数**: 在售产品数量

### 图表数据
- **销售趋势**: 最近30天的订单数和销售额
- **热销产品**: TOP 10产品及销量
- **分类销售**: 各分类销售额占比
- **地区分布**: 按省份统计订单数
- **实时销售**: 最近24小时销售曲线

### 表格数据
- **热销产品**: 排名、名称、型号、分类、销量、销售额、订单数
- **最近订单**: 订单号、客户、地址、金额、状态、时间
