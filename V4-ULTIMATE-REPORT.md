## 🚀 快速开始

### 开发环境

```bash
cd bearing-sales

# 初始化数据库
cd backend
node initDatabase.js
node scripts/createAdmin.js
node scripts/createSearchIndex.js

# 初始化供应链和CRM表
sqlite3 bearings.db < db/supply_chain.sql
sqlite3 bearings.db < db/crm.sql

# 启动后端
npm start

# 启动前端（Next.js）
cd ../frontend-next
npm install
npm run dev
```

### 生产环境（Docker）

```bash
# 配置环境变量
cp .env.production.example .env.production

# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f
```
