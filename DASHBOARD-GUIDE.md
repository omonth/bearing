# 后端可视化仪表板使用指南

## 概述

轴承销售系统现在包含一个功能强大的可视化管理后台，提供实时数据监控和分析功能。

---

## 🎯 功能特性

### 1. 实时数据统计
- ✅ 总订单数
- ✅ 总收入
- ✅ 平均订单金额
- ✅ 产品总数
- ✅ 增长趋势对比

### 2. 可视化图表
- ✅ **销售趋势图** - 最近30天的订单和销售额趋势
- ✅ **热销产品排行** - TOP 10热销产品横向柱状图

- ✅ **客户地区分布** - 按省份统计的订单分布
- ✅ **实时销售监控** - 最近24小时的销售曲线

- ✅ **热销产品详情** - 包含排名、销量、销售额等
- ✅ **最近订单** - 最新10条订单记录

### 4. 自动刷新
- ✅ 每30秒自动刷新数据
- ✅ 手动刷新按钮

## 📊 API 接口

### 综合仪表板数据
```http
GET /api/analytics/dashboard
```

返回所有仪表板数据，包括：
- 收入统计
- 热销产品
- 分类销售
- 客户分布
- 实时销售
- 销售趋势
- 最近订单

### 销售趋势
```http
GET /api/analytics/sales-trend?period=day&days=30
```

参数：
- `period`: 统计周期（day/week/month）
- `days`: 天数（默认30天）

### 热销产品
```http
GET /api/analytics/top-products?limit=10&days=30
```

参数：
- `limit`: 返回数量（默认10）
- `days`: 统计天数（默认30天）

### 分类销售统计
```http
GET /api/analytics/category-sales?days=30
```

### 客户地区分布
```http
GET /api/analytics/customer-distribution
```

### 收入统计
```http
GET /api/analytics/revenue-stats?days=30
```

### 实时销售监控
```http
GET /api/analytics/realtime-sales
```

## 🎨 界面说明

### 统计卡片
位于页面顶部，显示4个关键指标：
- **总订单数** - 显示订单总量和增长趋势
- **总收入** - 显示总销售额和增长百分比
- **平均订单金额** - 显示客单价
- **产品总数** - 显示在售产品数量

### 图表区域

#### 销售趋势图
- 折线图显示订单数量
- 柱状图显示销售额
- 支持鼠标悬停查看详情

#### 热销产品排行
- 横向柱状图
- 显示TOP 10产品
- 渐变色彩效果

#### 分类销售占比
- 环形饼图
- 显示各分类销售额占比
- 支持图例筛选

#### 客户地区分布
- 柱状图
- 按省份统计订单数
- 支持地区对比

#### 实时销售监控
- 面积图
- 显示最近24小时销售情况
- 平滑曲线展示

### 数据表格

#### 热销产品详情表
列：排名、产品名称、型号、分类、销量、销售额、订单数

#### 最近订单表
列：订单号、客户、地址、金额、状态、时间

## 🔧 自定义配置

### 修改刷新间隔

编辑 `dashboard.html`，找到：
```javascript
// 每30秒自动刷新
setInterval(loadData, 30000);
```

修改为你需要的间隔（毫秒）。

### 修改统计周期

修改API调用参数：
```javascript
// 获取最近7天的数据
const response = await fetch('/api/analytics/dashboard?days=7');
```

### 修改图表样式

在 `dashboard.html` 中找到对应的图表配置，修改 ECharts 选项：
```javascript
const option = {
  // 修改颜色
  color: ['#667eea', '#764ba2'],
  // 修改字体
  textStyle: {
    fontFamily: 'Arial'
  },
  // 其他配置...
};
```

## 🔍 故障排查

### 数据不显示
1. 检查后端服务是否启动
2. 检查浏览器控制台是否有错误
3. 检查数据库是否有数据

### 图表显示异常
1. 刷新页面
2. 清除浏览器缓存
3. 检查浏览器兼容性

### 数据不准确
1. 检查数据库数据
2. 检查API返回结果
3. 检查时区设置
