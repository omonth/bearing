# 分类标签

技能使用五个规范分类角色来表达。此文件将这些角色映射到此仓库问题跟踪器中使用的实际标签字符串。

| mattpocock/skills 中的标签 | 我们跟踪器中的标签 | 含义                           |
| -------------------------- | ------------------ | ------------------------------ |
| `needs-triage`             | `needs-triage`     | 维护者需要评估此 issue         |
| `needs-info`               | `needs-info`       | 等待报告者提供更多信息         |
| `ready-for-agent`          | `ready-for-agent`  | 完全指定，准备好 AFK 代理接手  |
| `ready-for-human`          | `ready-for-human`  | 需要人工实现                   |
| `wontfix`                  | `wontfix`          | 不会被执行                     |

当技能提到角色（例如"应用 AFK 就绪分类标签"）时，使用此表中对应的标签字符串。

编辑右列以匹配你实际使用的词汇。
